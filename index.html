Introduction to Web Development:

Web development is the process of creating websites and web applications. In today's digital world, it plays a crucial role in how we access information, communicate, and conduct business. Websites serve as the online face of individuals and organizations, making their development essential.

Web development is the process of creating, building, and maintaining websites and web applications.

Areas of Web Development:

Frontend Development:

Frontend development refers to everything that users see and interact with on the website.It involves the design, structure, and layout of the website and is often referred to as the ‘client side’ of an application.


Frontend Technologies:

    • HTML stands for HyperText Markup Language. It is the standard markup language used to create and design web pages, defining their structure and layout. 
    
    <!DOCTYPE html>
    <html>
    <head>
    <title>Page Title</title>
    </head>
    <body>

    <h1>My First Heading</h1>
    <p>My first paragraph.</p>

    </body>
    </html>
    • Cascading Style Sheets fondly referred to as CSS is a simply designed language intended to simplify the process of making web pages presentable. It is used to style our website.

    body {
    background-color: lightblue;
    }
    h1 {
    color: white; text-align: center;
    }
    p {
    font-family: verdana; font-size: 20px;
    }

    • JavaScript is a scripting language used to provide a dynamic behavior to our website.


    // store input numbers

    const num1 = parseInt(prompt('Enter the first number '));
    const num2 = parseInt(prompt('Enter the second number '));

    //add two numbers

    const sum = num1 + num2;

    // display the sum

    console.log(`The sum of ${num1} and ${num2} is ${sum}`);

Backend Development:

Backend development refers to the server side of a website, where the logic and data are processed and stored. Users do not directly interact with this part, but it ensures that the website works properly.

Core Server-Side Languages:

    • PHP is a widely used, open-source server-side scripting language primarily designed for web development.PHP code runs on the server (the computer hosting a website) to create the web pages you see.PHP works across different operating systems like Windows, Linux, and macOS.

                // Define an array containing

Full Stack Development:

Full-stack development refers to the practice of developing both the frontend and backend of a website or web application. Full-stack developers have a deep understanding of both areas and can build end-to-end solutions.

XAMPP:

XAMPP is a free program that lets you set up a mini web server on your computer. It includes everything you need to build and test websites, like a server (Apache), a database (MariaDB), and programming languages (PHP, Perl). Think of it as a personal practice ground for web developers, allowing them to create and fix websites on their own computer before putting them online for everyone to see.
